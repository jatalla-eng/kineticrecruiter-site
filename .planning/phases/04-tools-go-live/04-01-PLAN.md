**1. Create src/components/tools/ROICalculator.tsx (per D-08, D-09, D-10, D-11)**

`'use client'`. Imports: `useState`, `useMemo` from react; `getAllPlans` from `@/lib/plans`.

Read Professional tier price: `const proFee = getAllPlans().find(p => p.key === 'professional')?.monthly_price_cents ?? 5900;` — never hardcode price.

Input state (all numbers):
- `teamSize` (default: 2) — number of recruiters
- `placementsPerMonth` (default: 5)
- `timeToFillDays` (default: 30)
- `feePerPlacement` (default: 5000) — $ per placement

Calculations (useMemo, per D-11 formula assumptions):
```typescript
const hourlyRate = 50; // $50/hr average recruiter cost
const adminHoursPerPlacement = 4;
const timeFillReductionDays = inputs.timeToFillDays * 0.30;
const adminHoursSaved = inputs.placementsPerMonth * adminHoursPerPlacement * 0.40 * inputs.teamSize;
const timeFillHoursSaved = (timeFillReductionDays / inputs.timeToFillDays) * inputs.placementsPerMonth * 8 * inputs.teamSize;
const totalHoursSavedPerMonth = adminHoursSaved + timeFillHoursSaved;
const annualCostReduction = totalHoursSavedPerMonth * hourlyRate * 12;
const monthlyCost = proFee / 100; // cents to dollars
const paybackMonths = annualCostReduction > 0 ? (monthlyCost * 12) / annualCostReduction * 12 : 0;
const annualROI = annualCostReduction > 0 ? ((annualCostReduction - monthlyCost * 12) / (monthlyCost * 12)) * 100 : 0;
```

Layout: 2-column grid of number inputs (with labels), below that a 2x2 grid of result cards. Each result card: light teal bg (#E8F5F5), metric name, large teal number. Labels: "Time saved / month (hours)", "Annual cost reduction", "Payback period (months)", "Annual ROI (%)". Numbers formatted: hours as integer, cost as $X,XXX, payback to 1 decimal, ROI as integer %.

Use `type="number"` inputs with `min` values and `onChange` updating state.
