## Compliance Notes

## A/B Test Plan (Week 6 onward)

Once baseline data exists, run headline/description tests:

1. **Outcome vs Feature** — compare headlines leading with outcome ("Rank 200 CVs in 60 seconds") vs feature ("AI Resume Parsing") — hypothesis: outcome wins
2. **Specific vs Vague** — "7-Day Free Trial" vs "Free Trial" — hypothesis: specific wins
3. **Pain vs Aspiration** — "Stop Scrolling, Start Sourcing" vs "The AI ATS for Recruiters" — hypothesis: pain wins on cold traffic, aspiration on branded
4. **Urgency vs Evergreen** — "Try It This Weekend" vs "Try KineticRecruiter" — hypothesis: urgency minor lift only

Run each test 14 days minimum or 1,000 impressions per variant, whichever later.
